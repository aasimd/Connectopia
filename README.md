# Connectopia

## Social Media App to Connect People.

**Features:**

* Create a Post
* Follow/Unfollow User
* Add Images to create Posts
* Edit/Delete Post
* User Timeline
* Explore Feed
* Bookmark Post
* Delete Bookmark
* Authentication
* Like/Unllike
* User Profile (Bio, portfolio URL, Profile Picture)
* Edit User Profile
* Search User
